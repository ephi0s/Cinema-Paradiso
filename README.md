# Cinema Paradiso

App de ejemplo desarrollada con Vite y React para implementar un diseño básico de frontend y middleware. Para más info puedes ver [documentacion.md]

La implementación utiliza un archivo de JSON autogenerado que muestra las 500 películas más visitadas de IMDB, con dos vistas de la información, una en un visualizador de carátulas, donde carga la lista completa de películas, y una segunda autogenerada de detalles de película.

## Características:

- CSS modular y basado en variables
- Componentes modulares para las diferentes secciones y elementos reutilizables de la app
- Búsquedas básicas implementadas (por ahora, solo por título)
- Diseño _responsive_ para poder acceder desde dispositivos móviles
- Documentación del proceso de creación de la app
